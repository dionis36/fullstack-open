# Exercise 0.6: New note in Single page app diagram

```mermaid
sequenceDiagram
    participant browser
    participant server

    Note over browser: User types a note and clicks the Save button

    Note right of browser: JavaScript intercepts the form submission, prevents the default page reload, adds the new note to the local list, and redraws the notes

    browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa
    activate server
    Note right of browser: Request body (JSON): { "content": "...", "date": "..." }
    server-->>browser: HTTP 201 Created
    deactivate server

    Note right of browser: The browser continues running without reloading the page
```
